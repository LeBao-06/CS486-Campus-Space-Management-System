# 07-query-design Skill

**Objective:**
Design 5 meaningful SQL queries for answering business questions.

**Instructions for the Agent:**
1. Review the requirement in `req/business-requirement.md` and the schema in `outputs/05-db-definition-G11.sql`.
2. Create exactly 5 distinct, meaningful queries written for Microsoft SQL Server (T-SQL).
3. For each query, include comments explaining:
    - Business question being answered.
    - Target user(s) who would execute the query (e.g., Facility Manager).
    - A short explanation of why the query is useful.
4. Each query must be syntactically valid and runnable against the sample data. Queries might involve JOINs, aggregations, Subqueries, or CTEs.

**Output:**
Create or update `outputs/07-query-design-G11.sql`.
