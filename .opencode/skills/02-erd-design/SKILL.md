# 02-erd-design Skill

**Objective:**
Create a Conceptual Database Design (ERD) based on the business analysis from Task 1.

**Instructions for the Agent:**
1. Locate and read `outputs/01-business-req-analysis-G11.md`. Do not jump straight from the raw requirements.
2. Design the Entity-Relationship Diagram (ERD).
3. Identify main entities, attributes, relationships, cardinalities, and participation constraints.
4. Use Mermaid `erDiagram` block to represent the ERD (Crow's foot notation).
5. Add a brief description explaining your design choices and how they satisfy the identified business rules.

**Output:**
Create or update `outputs/02-erd-design-G11.md` containing the ERD and explanations.
